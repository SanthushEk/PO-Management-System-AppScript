# Purchase Order Management System

A basic **Purchase Order Management System** built for **practice purposes**.  
This project helps understand how purchase orders and vendor data can be managed digitally.

## Features
- Create purchase orders
- Manage vendor details
- View purchase order records
- Simple and easy-to-use interface

## Technologies Used
- Google Apps Script
- Google Sheets
- HTML
- CSS
- JavaScript

## Purpose
This project was created to practice Google Apps Script and basic system workflow concepts.
